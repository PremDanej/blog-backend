# Blog-Backend

### Get started

1. Clone this repository
   ``` requirements
   git clone https://github.com/PremDanej/RepoName
   ```

2. Go to the main folder -
   ``` commandline
   cd blog-backend
   ```

3. Install all dependencies -
   ``` requirements
   pip install -r requirements.txt
   ```

4. Run the project by
   ``` commandline
   python run.py
   ```
